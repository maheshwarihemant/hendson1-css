What is css?

  1. Css stands for cascading Style Sheets.
  2. Css describes how HTML elements re to be displayed on screen,paper,or in other media.

  3. Css is not neccery Css saves a lot of work. can control the layout of multiple web pages all at onces

  4. External stylesheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements!
  • Internal- by using a style element in the head section
  • External-by using a link element to link to an external CSS file.
  • The most common way to add CSS,is to keep the styles in external CSS files.Howerver,in This tutorial we will use inline and internal style,because this is casider to demonstrate,and easier for you to try it yourself.

When comparing CSS class vs ID,the difference is that CSS class applies a style to multiple elements.ID,on the other hand,applies a style to one unique element.ID is also spacial in that you can use a spacial URL to link directly to an element and it's used by javaScript.

In CSS,selectors are used to target a specific element or range of element on a web page.Once an element has been targeted,a style or set of styles can be applied to the element.

There is a wide range of selectors available.Two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied.
